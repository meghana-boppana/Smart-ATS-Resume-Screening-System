# Smart ATS Resume Screening System 🎯
**🚀 Live App:** [Click Here to Use](https://smart-ats-resume-screening-system-im6njnmjasmpl66ccqr23q.streamlit.app)
Automated Resume Screening System with JD-Base ATS Scoring, OCR Support, Admin Dashboard & Analytics | Python, Streamlit, SQLite
## 📋 Features
- OCR support for scanned resumes
- Admin Dashboard with analytics
- SQLite database for storing results
- PDF/DOCX resume upload
- ATS Score + Ranking + PDF Report
## 🛠️ 📄 Tech Stack
Python, Streamlit, PyMuPDF, Pandas, ReportLab, SQLite, Tesseract OCR
